# Application Blurb

## Short Version
Echo Drift is a Godot 4 2.5D exploration slice set on a suspended research ruin. I built it to demonstrate movement feel, spatial composition, collectible-driven progression, and atmospheric presentation in a compact portfolio-ready format.

## Resume Version
Built a Godot 4 exploration vertical slice featuring traversal, collectible scan recovery, and environmental storytelling, with emphasis on scene composition, player movement, and polished presentation for a game development portfolio.

## Interview Version
I wanted a project that felt more like a real game slice than a systems mockup. Echo Drift uses a small environment and a simple collectible objective to show judgment around scope, movement, composition, and finish. It is intentionally compact, but it still demonstrates how I structure scenes and turn a limited mechanic set into a polished application piece.
